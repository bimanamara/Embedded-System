IOT-based Smart Power Consumption Monitoring System using ESP32 and PZEM004T Sensor

Detail Proyek

Instrumen ini berfungsi untuk memonitoring parameter parameter listrik (arus, tegangan, daya, dan energi listrik terpakai) berbasis IoT yang akan membantu penghematan penggunaan listrik. Instrumen ini memiliki relay yang berfungsi untuk pemutus arus ketika pemakaian daya / konsumsi energi melebihi batas normal. Kemudian, terdapat 3 aktuator, yaitu LCD 16 X 2 untuk menampilkan parameter terukur, LED untuk representasi keadaan instrumen, dan Buzzer sebagai alarm.
